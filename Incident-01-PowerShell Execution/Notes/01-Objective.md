\# Incident 01: PowerShell Execution Detection



\## Objective



The objective of this incident is to detect and investigate the execution of a PowerShell process on a Windows 11 endpoint using Sysmon and Wazuh SIEM.



This exercise demonstrates how endpoint telemetry generated by Sysmon is collected by the Wazuh Agent, forwarded to the Wazuh Server, and analyzed through the Wazuh Dashboard.



The incident focuses on understanding process creation events, validating event collection, and performing an initial SOC investigation using real telemetry from the monitored endpoint.
